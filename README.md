# ChatItemSprite

**ChatItemSprite** is a lightweight Paper plugin that displays item sprites directly in Minecraft chat.

Instead of showing only plain text, items are rendered with their corresponding sprites, making chat messages more immersive and easier to understand.

## Features

* Display item sprites directly in chat
* Supports vanilla Minecraft items
* Supports custom items from **ItemsAdder**
* Lightweight and optimized for modern Paper servers
* Designed to work seamlessly with existing chat systems

## Requirements

* Paper 26.2+
* Java 25
